Computer Hardware: definition and components

MACRO-TOPIC: Hardware Setup & Safety

What is computer hardware?

Computer hardware is the physical part of a computer system: all the components that we can see and touch, such as the case, the motherboard, the processor, the memory and the external devices.

Main hardware components

  • CPU: processes all the instructions of the system.
  • RAM: fast memory that stores data temporarily.
  • Motherboard: main board that connects all components together.
  • Storage (HDD/SSD): stores the operating system, programs and user files.
  • Peripherals: keyboard, mouse, monitor, printer, etc.

What do we use it for?

Without hardware, operating systems and applications cannot run at all. Basic knowledge of hardware is essential for programming, PC installation and computer security.
