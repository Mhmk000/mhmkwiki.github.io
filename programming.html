MACRO-TOPIC 12: Programming Foundations

What is Programming?

Programming is the process of creating a set of instructions that tell a computer how to perform a task. These instructions are written in a programming language that both the human and the machine can understand.

The 5 Basic Elements of Programming

Regardless of the language, most programs use these five fundamental components:

  • Input: Getting data and commands into the computer.
  • Output: Getting results out of the computer (on screen, print, etc.).
  • Arithmetic: Performing mathematical calculations.
  • Conditional Execution: Checking certain conditions (If/Then) to decide what to do next.
  • Looping: Repeating a series of actions multiple times.

Programming Languages & Syntax

Each language has its own syntax (set of rules). Some common types are:

  • High-level languages: Easier for humans to read (Python, Java, JavaScript).
  • Low-level languages: Closer to machine code (Assembly, C).

The Importance of Comments

Comments are notes in the code that the computer ignores but are essential for programmers to understand what the code does.

In Java, we use:

  • // for single-line comments.
  • /* ... */ for multi-line comments.
