Operating Systems: Overview and Functions

MACRO-TOPIC: Operating Systems & Updates

What is an Operating System (OS)?

An operating system is a collection of software that interfaces between the human user and the hardware. It is the first software installed on a hard disk and the first to be loaded into memory when the computer is turned on.

The 5 Critical Tasks (The Conductor)

The OS acts like an orchestra conductor, managing five essential functions:

  • System Management: Managing the start-up (booting) and resources.
  • Communication: Managing connections to the Internet and email.
  • Security: Including utilities to keep viruses out and manage user authentication.
  • Hardware Services: Controlling peripheral devices (printers, monitors, etc.).
  • Software Services: Installing and managing applications (apps).

Categories and History

Operating systems have evolved significantly since the 1950s:

  • Desktop OS: Microsoft Windows (started with MS-DOS), Mac OS X (for Apple computers), and Linux.
  • Mobile OS: Designed for smartphones and tablets, with Android and iOS as the current leaders.

The OS Role in Security

The OS is the bedrock of computer security, maintaining three main pillars:

  1. Integrity: Protecting data from unauthorized changes.
  2. Availability: Ensuring systems work when needed.
  3. Confidentiality: Managing user identity and encryption.
