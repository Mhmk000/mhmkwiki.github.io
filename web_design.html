Web Design: basics and best practices

MACRO-TOPIC: Web Design (10)

Overview

This macrotopic introduces principles of visual design, responsive layouts, accessibility, and basic client-side technologies used to build modern web pages. It includes practical exercises for creating a small website prototype and guidelines for presenting the work.

Learning objectives

  • Understand HTML structure and semantic tags.
  • Apply CSS for layout, typography and responsive behaviour.
  • Make pages accessible and mobile-friendly.
  • Prepare a small project demonstrating a web landing page.

Activities

  • Create a simple multi-section landing page using semantic HTML.
  • Implement a responsive grid and a navigation menu.
  • Validate accessibility using basic tools and manual checks.
  • Package the project and include a short explanatory README.

Resources

Place the provided resource file in the site folder (suggested name: MACRO-TOPIC-10-WEB-DESIGN.pdf) to allow local consultation, or link to external tutorials on HTML, CSS and responsive design. If the resource file is present, it will be available for students.
