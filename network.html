What is a computer network?

A computer network connects two or more devices so they can exchange data and share resources. Networks range from small local area networks (LANs) to wide area networks (WANs) and the Internet.

Network types

  • LAN: local area network for homes and offices.
  • WAN: connects multiple LANs over large distances.
  • Wi-Fi: wireless local network based on IEEE 802.11 standards.
  • Peer-to-peer: simple networks where each device can act as client and server.
  • Client-server: networks with dedicated servers providing services to clients.

Common network incidents

  • No connectivity: device cannot reach the network or Internet.
  • Slow performance: high latency or low bandwidth.
  • IP conflicts: two devices with the same IP address.
  • DNS failures: domain names do not resolve to IP addresses.
  • Authentication issues: unable to join secure networks.

Basic troubleshooting steps

  1. Check physical connections (cables, Wi‑Fi status).
  2. Restart the device and the router/switch.
  3. Verify IP configuration with `ipconfig` / `ifconfig`.
  4. Test connectivity with `ping` and `tracert`/`traceroute`.
  5. Check DNS settings and try an alternate DNS server.
