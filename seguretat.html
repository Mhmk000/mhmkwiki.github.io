Computer Security: definition and measures

MACRO-TOPIC: Hardware Setup & Safety – Security

What is computer security?

Computer security is the set of controls and measures that protects computers, networks and data from unauthorized access, theft, misuse or damage. [web:2][web:3][web:6]

It usually focuses on three main goals: confidentiality (only authorised people can access data), integrity (data cannot be changed without control) and availability (systems are working when needed). [web:3][web:9]

Security components

  • Physical security: protecting hardware (PCs, servers, routers) from theft, damage or tampering. [web:3][web:7]
  • Network security: firewalls, VPNs and other mechanisms to control traffic between internal and external networks. [web:5][web:6][web:11]
  • Software and data security: antivirus, updates, backups and data encryption. [web:2][web:5][web:8]
  • Policies and procedures: strong passwords, user roles, access policies and user awareness training. [web:3][web:9]

Basic security measures

Some simple measures that can be applied on any computer or small network:

  • Keep the operating system and applications updated to fix vulnerabilities. [web:2][web:6]
  • Install antivirus and anti‑malware software and keep them up to date. [web:2][web:5]
  • Enable and correctly configure the firewall. [web:2][web:6][web:8]
  • Make regular backups of important data. [web:5][web:8]
  • Use strong passwords and, when possible, multi‑factor authentication. [web:3][web:11]
  • Protect equipment physically (locks, labels, restricted areas). [web:3][web:7]
