Computer Security: definition and measures

MACRO-TOPIC: Safety & Security

What is computer security?

Computer security is the set of controls and measures that protects computers, networks and data from unauthorized access, theft, misuse or damage.

It usually focuses on three main goals: confidentiality (only authorised people can access data), integrity (data cannot be changed without control) and availability (systems are working when needed).

Security components

  • Physical security: protecting hardware (PCs, servers, routers) from theft, damage or tampering.
  • Network security: firewalls, VPNs and other mechanisms to control traffic between internal and external networks. ]
  • Software and data security: antivirus, updates, backups and data encryption.
  • Policies and procedures: strong passwords, user roles, access policies and user awareness training.

Basic security measures

Some simple measures that can be applied on any computer or small network:

  • Keep the operating system and applications updated to fix vulnerabilities.
  • Install antivirus and anti‑malware software and keep them up to date.
  • Enable and correctly configure the firewall.
  • Make regular backups of important data.
  • Use strong passwords and, when possible, multi‑factor authentication. ]
  • Protect equipment physically (locks, labels, restricted areas).
